# 图片采集与补充清单（知识库管理）

本清单用于统一管理本手册所需的截图采集与落盘，确保“先文字可用，后统一补图”。

## 约定
- 保存路径：docs/使用说明/images/
- 文件命名：中文名，场景+序号，例如：文档分段_1.png
- 标注规范：
  - 浏览器 100% 缩放，窗口≥1440×900
  - 目标元素加 4px 红色描边（#FF0000），必要时箭头/标题
  - 仅保留必要 UI 区域，避免干扰元素
